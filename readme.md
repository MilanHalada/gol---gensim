# GOL - GenSim

A grid-based ecosystem simulation written in [Odin](https://odin-lang.org/), rendered with [Raylib](https://www.raylib.com/).

Cells evolve through four types - empty, resource, grazer, and predator - with energy, feeding, reproduction, and mutation. Resources grow in open space, grazers consume resources, and predators hunt grazers, producing emergent population dynamics on a toroidal grid.
